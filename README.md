# doEcommerceTag

抖店商品上架页批量创建自定义规格 Chrome 插件项目。

## 目标

用于在抖店商品发布 / 编辑页面中：
- 导入每日 Excel
- 解析「直播间编号 -> 店铺编号」映射
- 在“颜色分类”规格类型下批量创建直播间编号规格值
- 自动填充对应店铺编号 / 商家编码
- 保留人工最终检查与手动发布
